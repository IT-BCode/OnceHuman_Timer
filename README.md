## 🛠 사용 방법
1. **타이머 추가:** 좌측 상단의 '위젯 추가' 버튼을 눌러 새 타이머를 생성합니다.
2. **시간 설정:** 타이머 주기를 설정하고 '시작'을 누릅니다.
3. **공유:** 타이머 공유 '설정'버튼을 눌러 데이터 공유 링크를 생성할 수 있습니다.
4. **워터마크:** '워터마크 모드'를 활성화하면 게임 화면 방해 없이 남은 시간을 체크할 수 있습니다.
